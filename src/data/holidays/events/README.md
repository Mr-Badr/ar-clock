# Holidays Events

Each event owns one authoring folder:

```text
src/data/holidays/events/<slug>/
  package.json
  research.json
  qa.json
```

Use [`docs/add-new-event.md`](../../../../docs/add-new-event.md) as the authoritative guide for:
- how to scaffold a new event
- what every JSON key means
- how to build and validate
- what generated files must never be edited by hand
