# ⏳ حاسبة العمر الشاملة — الخطة الاحترافية الكاملة
## Ultimate Arabic Age Calculator — Full Professional Blueprint

> **Stack:** Next.js 16 App Router · shadcn/ui · Tailwind CSS · RTL · PWA · ISR · Suspense  
> **CSS Rules:** كل قيم التصميم مُرجَعة لـ `app/styles/` حصراً — لا hard-coded colors أو spacing أو typography  
> **الهدف:** أول أداة حساب عمر عربية متكاملة تتفوق على TimeAndDate وكل المنافسين العرب

3. لا مشاهير عرب يشاركون نفس يوم الميلاد
4. لا "ماذا حدث في التاريخ الإسلامي يوم ميلادك"
5. لا حاسبة تقاعد حسب الدولة العربية
6. لا مخطط "كيف تتقدم أنت وشريكك في العمر معاً"
7. لا حاسبة عيد الميلاد النصفي
8. لا تحديد الجيل (Gen Z / Millennial / Gen X) بالسياق العربي

```
- المعادلة: العمر الهجري = العمر الميلادي × 1.0307
```

#### H2: كيف تُحسب الأعمار بدقة؟
```

مع مثال محلول خطوة بخطوة
```

#### H2: معلومات شيقة عن عمرك
```
- في أي فصل ولدت؟
- ما هو جيلك؟ (جيل Z / ميلينيال / جيل X)
```

## 🎨 نظام التصميم العام

### المبدأ الأساسي: "وضوح أولاً، جمال ثانياً"

كل صفحة في هذه الحاسبة تتبع:
1. **Input فوري** — لا Wizards طويلة
2. **نتيجة على الفور** — بدون انتظار
3. **بصريات ذكية** — Charts وSVG حيث تضيف قيمة
4. **لا text dump** — كل رقم مُعروض بسياق بصري
5. **CSS من `app/styles/`** فقط — لا قيم hard-coded

### قواعد التصميم لكل صفحة

```
Desktop: Input يسار (sticky) · Results يمين (sticky)
Tablet: Input أعلى · Results تحته
Mobile: Input → Results في نفس الـ flow (لا sticky)

الـ Hero: تدرج خفيف من tokens.css · لا gradients ثقيلة
Cards: border خفيف + خلفية secondary · radius من tokens
Charts: ألوان من design tokens · لا hard-coded hex
Typography: كل الأحجام من tokens · لا px مباشرة
```

### Shadcn المستخدمة

```
Inputs:
- Select (dropdown الشهر والسنة)
- RadioGroup (هجري/ميلادي)
- Input (إدخال السنة مباشرة)
- DatePicker مخصص (RTL-safe — مبني على shadcn Calendar)

- Slider (للإنجازات المستقبلية)

- Progress (Progress Bars للإنجازات)

- HoverCard (معلومات إضافية عند hover)
- Accordion (FAQ)

Feedback:
```

## 🔧 Performance — PWA / Cache / Suspense

> كل هذه التقنيات موجودة في التطبيق — الخطة تُشير إليها فقط

- **PWA:** آخر نتيجة حساب تُخزَّن في `localStorage` → تعمل offline
- **ISR:** المحتوى التعليمي والبيانات الثابتة: `revalidate: 86400`
- **Suspense:** كل Calculator Component مُغلَّف بـ `<Suspense fallback={<AgeCalculatorSkeleton />}>`
- **Cache:** نتائج الحساب تُخزَّن بـ `URL params` → Shareable Link
- **Skeleton:** لكل section (InputSkeleton, ResultSkeleton, ChartSkeleton)
